# JLM Projects

This folder contains all JLM-related work. Each subfolder has its own `CLAUDE.md` with specific context.

## Project Areas

| Folder | Purpose | When to Read Its CLAUDE.md |
|--------|---------|---------------------------|
| `jlmops/` | Backend operations (Google Apps Script) | Working on backend code, sync, tasks, dashboard |
| `website/` | Frontend website code | Working on site UI, customer-facing features |
| `content/` | Blog posts, guides, video scripts | Writing or editing content |
| `business/` | Strategy documents | Brand voice, content strategy, planning |
| `marketing/` | Campaigns, promotions | Email campaigns, ads, customer communications |

## Quick Orientation

**What is JLM Wines?**
Online wine retailer in Israel. Anti-snob positioning—makes wine accessible, convenient, and easy. No jargon, no pretension.

**Brand voice (for anything customer-facing):**
- Friendly, personal, never talks down
- Polite and earnest, never negative about competitors
- Goal: help people get wine they'll enjoy without making them feel dumb

**Full brand guidelines:** `business/CONTENT_STRATEGY.md`

## Utility Folders

- `docs/` — General documentation
- `exchange/` — File exchange/temp storage
- `.claude/` — Shared commands, wishlist, bugs (see `jlmops/CLAUDE.md` for details)

## Starting a Task

When directed to work on a specific area:
1. Read that folder's `CLAUDE.md` first
2. Don't front-load context from other folders unless needed
3. Follow folder-specific conventions
