# JLM Wines — Brand Standards Index

**Purpose.** Ensure consistent voice, tone, and visual identity across all customer-facing channels. This document is an index that points to the existing per-area docs and identifies where standards are missing.

**Cadence.** Reference doc, rarely changes. Update only when a brand decision shifts.

---

## Voice

**Friendly, personal, never talks down. Anti-snob. Customer-first.**

| Domain | Doc |
|--------|-----|
| Full strategy | `business/CONTENT_STRATEGY.md` |
| Content quick-reference | `content/CLAUDE.md` |
| Email tone | `marketing/EMAIL_GUIDELINES.md` |
| Marketing tone | `marketing/CLAUDE.md` |

**Core rules (consolidated):**
- Write like a trusted friend who happens to know wine — not a sommelier, not a salesperson, not a teacher
- People who drink wine, not "wine lovers"
- Goal: help people get wine they'll enjoy without making them feel dumb
- Polite, earnest, upbeat — never negative about competitors, brands, wineries, or grapes
- No jargon, no pretension, no condescension
- Sensory framework (intensity / complexity / acidity) is an internal tool — customers should never feel they need to learn it
