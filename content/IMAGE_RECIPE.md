# JLM Wines Blog Featured Image Recipe

## Purpose
Canva AI prompts for blog featured images:
- Blog roll thumbnails
- Post headers (horizontal strip with semi-transparent text overlay)
- Social media, newsletter, email

## Visual Style
**Impressionist oil painting** — painterly, soft-focus, classy
- Contrasts with website's crisp/commercial look
- Signals editorial content vs product photography

---

## PROMPT FORMULA

```
Impressionist oil painting, close-up of [OBJECT], soft dreamy background, [COLOR TONES], [LIGHTING], [MOOD], 16:9 landscape
```

### Elements
| Element | Purpose | Examples |
|---------|---------|----------|
| **Style** | Painterly feel | "Impressionist oil painting" |
| **Composition** | Tight focus | "close-up of" |
| **Background** | Text overlay friendly | "soft dreamy background" |
| **Color tones** | Match content | burgundy/amber for reds, citrus/gold for whites |
| **Lighting** | Mood setting | "golden hour" (warm), "bright natural" (fresh) |
| **Mood** | Emotional tone | 2-3 words: "intimate and inviting" |
| **Format** | Blog header | "16:9 landscape" |

### Object Types by Category
| Category | Object Focus |
|----------|--------------|
| **Educational** (guides, attributes) | Wine elements: pour, glass, swirl, cork, bottle |
| **Trust/Selection** | Curation: bottles arranged, shelves, hand with glass toward sun |
| **Practical** (pairing, storage) | Context: food pairings, wooden rack, table settings |
| **Foundational** | Warm scenes: glass catching light, seasonal elements |

### Notes
- Canva naturally avoids people/hands/ice when not mentioned
- Avoid negative prompts ("no X") — they produce the opposite
